关于安装双系统差点把博客搞没这件事

如本次标题所说，这次心血来潮，想给我的laptop整个双系统，黑苹果暂时没考虑，选择了win+Linux双系统，然后这次选择的是Ubuntu20.04.1，也怪我，当时博客专门分了个盘，然后搞没了，哈哈哈哈哈…

解决方案:(￣▽￣)” 幸好当时备份了博客，就是少了篇文章，就是这篇链接直达

这个博客是基于hexo+Butterfly+github搭建的，图库用的是jsdeliver。其实不用将所有的文件都还原(这个方法换电脑照样适用)。

只需要拷贝一下目录

  • _config.yml
  • package.json
  • scaffolds/
  • source/
  • themes/

然后把这些文件放到一个目录下，例如hexo(就是一个文件夹)

然后再配置hexo的环境：安装node

具体安装方法网上都有

安装完node之后，再执行以下命令

npm install -g hexo

安装好执行完命令后，进入上面创建的hexo目录中

准备:这里win+R用cmd也行，用Git也行(建议用Git)

模块安装，执行命令

  • npm install
  • npm install hexo-deployer-git –save
  • npm install hexo-generator-feed –save
  • npm install hexo-generator-sitemap –save

最后就是部署，然后执行命令

  • hexo g
  • hexo deploy

以上就是本次的手残操作，还有补救就很棒，要不就得重新搭载了，嘤嘤嘤，大难不死，必有后福，真好，又水了一篇，下次打算出个《从0开始搭建属于自己的博客》，再好好巩固一下
